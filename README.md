# 48Live

# 最近某些直播源换到阿里云了，不允许跨域
# 本来打算用反向代理的，但是服务器带宽不够- -

   目前只在Chrome下测试通过

## 项目参考 [pocket48-grab](https://github.com/xsaiting/pocket48-grab/)
   接口以及成员信息都从这里来的

## 基于Laravel + Vue

## 安装

```bash
composer update
npm install
npm run dev
```
